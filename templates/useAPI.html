Guide: How to use this API

This API will give you information about IPL

want to know how many teams played use this

127.0.0.1:5000/api/teams

want know team VS team information use this

127.0.0.1:5000/api/teamVteam?team1={1st team name}&team2={2nd team name}

want to know team records

127.0.0.1:5000/api/team-record?team={team name}

want to know batter record

127.0.0.1:5000/api/batter?batsman={batsman name}

want to know bowler record

127.0.0.1:5000/api/bowler-record?bowler={bowler name}
